Quem somos.

We've been designing and developing software for open source content management systems (CMS) for over 6 years. Our company, YOOtheme, is a very popular theme club for Joomla and WordPress. We have great experience in building themes, extensions and plugins and we know WordPress and Joomla by heart. Over the last years we've noticed that our beloved CMS's can't keep up with modern web application trends.

What's lacking.

Popular CMS's like WordPress, Joomla and Drupal have grown over the time and it is hard for them to adapt to latest web technologies. They are feature heavy, bloated and have to stick to legacy code to keep compatibility with their large extension ecosystem. It will take a lot of work for them to implement modern design patterns and coding standards like today's popular PHP 5.3+ frameworks do.

A fresh start.

Pagekit is a new CMS we are developing right now. It is built from the ground up with a modern architecture in mind. We took the good parts of existing CMS's and mixed them with new ideas. Pagekit will also serve as a web application framework and will provide an awesome platform for theme and extension developers. It's a powerful, modular and lightweight CMS focused on simplicity.

A platform to
create and share

Everything in Pagekit is modular and extendible by default. This makes Pagekit the perfect platform to build your own web application. In fact, we want Pagekit to be the best platform for a thriving extension and theme ecosystem. That's why Pagekit will feature a built-in marketplace from the beginning. We aim to attract a large number of developers by providing the services they need. Developers can distribute their extensions and themes instantly to a global community of users. The marketplace supports both open-source and commercial extensions. Update notifications, one-click updates and a well thought-out package manager are already in place.

Technology
that will excite you

Pagekit with its web application framework is the right choice for beginners and also for experienced developers. It provides a set of components that solve common tasks in content management with a unified API. The dependency injection container offers great flexibility, as service classes get injected at run-time and can easily be changed. Pagekit uses Symfony's HTTP and routing components, which provide a solid and well tested solution for request and response handling. The database layer is based on Doctrine and is combined with Pagekit's very own object relational mapper that offers a fast and lightweight implementation for object persistence. Pagekit follows the PSR-1 coding standard and leverages powerful features like class autoloading, namespaces, annotations and closures.

Simple User Interface

Pagekit is focused on simplicity. The user interface will reflect that by being clean, intuitive and efficient. We won't bloat it with unnecessary features. It will have a beautiful, clutter-free look. The Pagekit administration will be built with UIkit - a lightweight and modular front-end framework. This makes the administration fully responsive by default. The whole functionality will work seamlessly on any mobile or touch device.

Your site's CSS and JavaScript is completely up to you. You choose your favorite front-end framework or just craft everything by hand. No dependencies! You can update Pagekit and your site stays intact.

Key features at a glance

Markdown editor

Markdown editors are all the rage. Pagekit comes with one that is well thought-out and extendable, providing a great writing experience. We love it and believe that you will love it too.

Template engine

Pagekit supports not only pure PHP templating, but also Razr, its own lightweight templating engine, whose simple and expressive syntax makes templates more readable and fun to type.

Users & Permissions

Our permission system is easy to use with permissions that have speaking names and control exactly what users may or may not do on your site. There's no tricky ACL with inheritance.

URLs & Routing

The default URLs used are already descriptive and easy on the eye. However, Pagekit uses URL aliases that allow you to change each URL to anything you want.

Open Source
and MIT licensed

Yes, Pagekit is completely free and you can do anything you want with it. No restrictions! You have the choice, how you license your themes and extensions for Pagekit. You can use the MIT, GPL or any other license. This gives you as a developer REAL freedom!

Pagekit is hosted on GitHub and will be open for everyone to contribute. We are looking forward to all the contributions and a growing community.

So, you probably want to know...

Who are you?

YOOtheme is a very popular theme club for Joomla and Wordpress. Founded in 2007, we have over 6 years expierence in building themes, extensions and plugins for Joomla and WordPress.

What about Joomla and WordPress?

We love them both! Of course, we will continue to develop themes and extensions for them. In fact, our Warp theme framework is platform independent and we will just add support for Pagekit.

Will themes be available with the initial release of Pagekit?

Yes, with this concept page we are looking for partners, so themes and extensions will be available from the beginning. YOOtheme will make its Warp framework compatible with Pagekit so there will definitely be many themes available.

How far along is the development progress?

Pagekit has been under development since 2012. Third party frameworks are picked and the Pagekit foundation and API is also very stable. We are already working on the user interface and features. Here is a blog post about how it all started.

When will Pagekit be released?

We hope to have an alpha version for our partners by March of 2014. The public release is planned for early summer 2014.

Will the initial release be ready for production?

Yes, Pagekit will be fully functional and ready for production. It will have everything you need to build blogs and websites.

What about the marketplace?

Yes, there will be a built-in marketplace for free and commercial themes and extensions similar to the Apple app store with reasonable conditions for developers.
